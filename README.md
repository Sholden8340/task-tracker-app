# Task tracker application

```git clone```

```cp .env.example .env```

```composer install --ignore-platform-reqs```

```composer require laravel/sail --dev```

```php artisan sail:install```

```./vendor/bin/sail npm ci```

```./vendor/bin/sail npm run build```

```./vendor/bin/sail up -d```
